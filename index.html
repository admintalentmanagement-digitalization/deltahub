What DeltaHub Can Do

A safer way to update Working Master data with clear controls, exception reports, and audit trails.

Smart Lookup & UpdateUpdate thousands of records without manual VLOOKUP. Match by NIK or any selected key column.
Formula Logic EngineApply configurable business rules such as Active SAP, Potential Non Active, Talent tagging, and custom logic.
Exception CenterDetect blank keys, duplicate conflicts, missing references, and strict mapping mismatches before applying updates.
Downloadable Error ReportsDownload only problematic rows so users do not need to search through thousands of records manually.
Audit Trail & Change HistoryTrack before-after values, source files, update timestamps, and applied change counts.
Multi-Step Data IntegrationConsolidate Employee, Talent, PAT, Succession, and custom updates into one Working Master.
Safe Apply ProcessPreview all detected changes first. Nothing is written to the Working Master without user confirmation.
Token-Safe AIAI assistance is optional and only runs when triggered by the user. Full employee rows are not sent automatically.

Catatan:

Aturan Working Master:
  • Working Master menjadi base data project dan akan dikunci setelah diupload.
  • Untuk menggunakan master berbeda, pastikan seluruh update sudah di-apply ke Working Master dan Final Excel sudah di-download, kemudian buat Project Baru.
  • Jika ingin menambah kolom baru atau mengubah nama kolom, lakukan pada file master hasil download lalu buat Project Baru.
  • Kolom yang akan diupdate harus sudah tersedia di Working Master.
  • Nama kolom Working Master dan Reference harus sama persis.
  • Duplicate key dengan nilai berbeda tidak diperbolehkan.
